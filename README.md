# Oriz Grid — QR + barcode tools

> Generate, decode, and customise QR codes and barcodes. The brand **Grid** comes from the QR code's signature grid pattern.

**Live at**: https://qr.oriz.in · **Status**: scaffold

## What this is

A browser-based QR and barcode toolkit. Generation and decoding will run entirely in your browser — codes never travel over the network.
