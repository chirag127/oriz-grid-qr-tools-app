# Oriz Grid — QR + Barcode Tools

Generate, decode, and customise QR codes and barcodes. The product brand **Grid** comes from the QR code's signature grid pattern.

Live at **https://qr.oriz.in**.

## Stack

- [Astro 6](https://astro.build/)
- [React 19](https://react.dev/) (islands)
- [Tailwind CSS v4](https://tailwindcss.com/)
- [qrcode.js](https://github.com/soldair/node-qrcode)
